# Earth Guardian - 환경 뉴스 블로그

지구 환경 관련 최신 뉴스를 자동으로 수집하여 보여주는 웹 애플리케이션입니다.

## 기능

- 다양한 환경 뉴스 소스의 RSS 피드를 자동으로 수집
- 최신 뉴스를 날짜순으로 정렬하여 표시
- 반응형 디자인으로 모바일/데스크톱 모두 지원
- 뉴스 이미지 자동 표시

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
python app.py
```

3. 웹 브라우저에서 다음 주소로 접속:
```
http://localhost:5000
```

## 기술 스택

- Python
- Flask
- Feedparser
- BeautifulSoup4
- Tailwind CSS
